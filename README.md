# Proyecto Casillas-Seguras 
## Avances Recientes (Rama: dev-kathy)

En esta etapa del desarrollo, me enfoqué en sentar las bases de la seguridad y la gestión de identidades para la aplicación.

### 1. Implementación del Modelo de Usuario
Se creó la clase `Usuario` y se aplicaron los siguientes conceptos:
* **Encapsulamiento:** Atributos privados (`username`, `password`, `rol`) para proteger la información.
* **Lógica de Validación:** Implementación del método `validarCredenciales` para permitir el acceso al sistema.
* **Métodos de Acceso:** Creación de *Getters* para obtener información del usuario de forma segura
